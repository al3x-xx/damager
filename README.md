![](png.png)\
блестхак: https://www.blast.hk/threads/60774/
### Commands:
* /dmg - on/off
* /dmg pos - сменить позицию меню
* /dmg menu - открыть/закрыть imgui меню или клавишей F2 по умолчанию
* /dmg key value - клавиша активации
* /dmg key - режим активации Pressed или Down
* /dmg check - проверка игрока по статусу или по хп
* /dmg afk - дамажить игроков в афк on/off
* /dmg mode value - режим дамагера (0 - по id, 1 - по хп, 2 - по дистанции, 3 - по пингу, 4 - по fov, куда направлена камера)
* /dmg onv - дамажить только тех кто на экране on/off
* /dmg fov value - сменить fov для 4-го мода
* /dmg frac value - выбор фракции /dmg frac none - никого, /dmg frac all - по всем, /dmg frac aztec rifa ballas etc - по фракциям
* /dmg af - дамажить всех кроме выбранных фракций on/off
* /dmg as - авто выставление настроек on/off
* /dmg sfs - сохранение/загрузка настроек под текущий сервер on/off
* /dmg ra - уменьшение патрон on/off
* /dmg gun id - выбор с какого оружия дамажить
* /dmg delay value value2 - задержка, /dmg delay 100 - статическая, /dmg delay 100 200 - рандом от 100 до 200
* /dmg radius value - радиус
* /dmg miss value value2 - промахи, /dmg miss value - статические, /dmg miss value value2 - рандом от value до value2
* /dmg miss - промахи on/off
* /dmg nokill value - не будет дамажить если хп игрока ниже указанного
* /dmg nokill - нокилл on/off
* /dmg bl id - добавить в черный список, сначала будет дамажить черный список, а после перейдет на фракцию
* /dmg dbl value - убрать из черного списка
* /dmg dbl - убрать всех из черного списка
* /dmg ig id - добавить в игнор список
* /dmg dig value - убрать из игнор списка
* /dmg dig - убрать всех из игнор списка
