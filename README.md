### Commands:
* /dmg - вкл/выкл
* /dmg pos - сменить позицию менюшки (нажать лкм для сохранения)
* /dmg menu - открыть менюшку(imgui) | или клавишей F2 по дефолту
* /dmg key value - кнопка активации (вместо value id кнопки)
* /dmg key - режим активации Pressed или Down
* /dmg check - проверка жив ли игрок по статусу или по хп (если дамажит мертвого ставьте по хп)
* /dmg afk - дамажить афкшников или наоборот
* /dmg mode value - режим дамагера (0 - по id, 1 - по хп, 2 - по дистанции, 3 - по пингу, 4 - по fov т.е куда смотрите)
* /dmg onv - дамажить тех кто на экране или наоборот
* /dmg fov value - сменить fov (для 4-ого мода)
* /dmg frac value - выбор фракции (/dmg frac none - никого, /dmg frac all - по всем, /dmg frac aztec rifa ballas и т.д - по фракциям, фракции берутся с файла, так шо можно добавлять свои или изменять существующие)
* /dmg af - вкл/выкл антифракцию (дамажит всех кроме выбранной фракции)
* /dmg as - вкл/выкл авто выставление настроек для оружий
* /dmg sfs - вкл/выкл сохранение/загрузку настроек под текущий сервер
* /dmg ra - вкл/выкл уменьшение патрон
* /dmg gun value - выбор с какого оружия дамажить (вместо value id оружки, например стреляете с дигла, а дамажите с какого выбрали)
* /dmg delay value value2 - задержка (/dmg delay 100 - статическая, /dmg delay 100 200 - рандом от 100 до 200)
* /dmg radius value - радиус
* /dmg miss value value2 - промахи (/dmg miss 4 - статические /dmg miss 4 7 - рандом от 4 до 7)
* /dmg miss - вкл/выкл промахи
* /dmg nokill value - нокилл, не будет дамажить если хп игрока ниже указанного (вместо value хп)
* /dmg nokill - вкл/выкл нокилл
* /dmg bl id - добавить в черный список (сперва будет дамажить черный список, а после перейдет на фракцию)
* /dmg dbl value - убрать из черного списка (вместо value позицию ника)
* /dmg dbl - убрать всех из черного списка
* /dmg ig id - добавить в игнор список (не будет дамажить игроков которые в игнор списке)
* /dmg dig value - убрать из игнор списка (вместо value позицию ника)
* /dmg dig - убрать всех из игнор списка

больше настроек в менюшке\
по дефолту все уже настроено, остается выбрать фракцию и зажать кнопку R
